== Overview

Spring Session BOM provides dependency management for Spring Session modules.

Importing the BOM allows applications to use consistent Spring Session dependency versions without specifying versions for each module individually.

== Usage

=== Maven

[source,xml]
----
<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.session</groupId>
            <artifactId>spring-session-bom</artifactId>
            <version>${spring-session.version}</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>
----

=== Gradle

[source,groovy]
----
dependencyManagement {
    imports {
        mavenBom "org.springframework.session:spring-session-bom:${springSessionVersion}"
    }
}
----
